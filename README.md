Clone the nodejs-webhook-server-sample.\n
Install node using home brew.\n
Install npm into nodejs-webhook-server-sample\n
npm run dev

npm install hookdeck-cli -g\n
Create hookdeck events::\n
hookdeck listen 1337\n
